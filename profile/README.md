# EGS Survey Australia — Python Scripts

This organisation is home to Python tools developed by the EGS Survey Australia team for survey data processing, analysis, and reporting.

## Getting started

All tools in this organisation use **Miniforge** for environment management. Before cloning any repository, set up Miniforge on your machine.

→ **[Miniforge Setup Guide](../MINIFORGE_SETUP.md)**

Once Miniforge is installed, every tool follows the same pattern:

```bash
git clone https://github.com/EGS-Survey-Australia/<repo-name>.git
cd <repo-name>
conda env create -f environment.yml
conda activate <env-name>
```

The `environment.yml` file in each repo handles all dependencies automatically.
